Mobile First Design

I don't like writing code twice and so when it comes to mobile apps and websites my first priority is User Experience and Frameworks. For web based solutions I have gravitated towards React and Angular for client side scripting and seperating out my API's into there own solutions. It not only makes it easier for multiple developers but it also breaks out the seperation of duties and creates a clear objective for both the client and the server.

For mobile app development my go to has been Xamarin. Not only because of the great ability to develop in C# but also due to the fact that I can share my code across Android and iOS. For mobile web applications my go to has been Angular 6 with Bootstrap 4.
